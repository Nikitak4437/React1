What is Css?

  1. CSS stands for Cascading Style Sheet.

  2. CSS describes how HTML elements are to be displayed on screen, paper, or in other media.

  3. CSS is not necessary CSS saves a lot of work. it can control the layout of multiple web pages all at once.

  4. External stylesheets are stored in CSS files

CSS can be added to HTML documents in 3 ways.

  • Inline - by using the style attribute inside HTML elements1

  • Internal - by using style element in the head section.

  • External - by using the link element to link to an external CSS file.

  • The most common way to add the css, is to keep the styles in external CSS files. However in this tutorial we will use inline and internal styles, beacuse this is easier to demonstrate, and easier for you to try it yourself.

When comparing CSS class vs ID, the difference is that CSS class applies a style to multiple elements. ID, on the other hand, applies a style to one unique element. ID is also special in that you can use a special URL to link directly to an element and it's used by JavaScript.

In CSS, selectors are used to target a specific element or range of elements on a web page. Once an element has been tergeted, a style or set of styles can be applied to the element.

There is a wide range of selectors available. Two of the most common used are class and ID. Both are used to target elements to which a style should be applied.
